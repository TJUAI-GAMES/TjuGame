牟婧妍：
      1.人类进化的过程中，游戏曾经作为人类生存技能的训练项目。
      2.人类基因中有游戏的部分。
      3.现代生活中，生存技能类的游戏已经渐渐消失，更多的是纯供娱乐的游戏。

王志勇:

	1.人类游戏是因为人们在游戏中得到反馈/奖励/回报。
	
	2.人类游戏是因为游戏中存在获得正反馈的机会。
